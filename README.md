## Overview
Coder Detail is a developer-focused platform where programmers can create profiles showcasing their coding skills and achievements. The platform integrates with popular coding platforms like Codeforces and LeetCode, displaying users' ratings and performance metrics. Visitors can browse developer profiles and sort them by various parameters including rating, or platform-specific metrics, making it easy to discover top talent or find peers with similar skills.

## Features
- **User Profiles**: Create and customize your developer profile with skills, and portfolio
- **Platform Integration**: Connect with Codeforces, LeetCode.
- **Performance Metrics**: Showcase your ratings and achievements from connected platforms
- **Discovery System**: Find developers based on skills, ratings, or specific platform performance
- **Skill Visualization**: Visual representation of your technical skills and proficiency levels
- **Responsive Design**: Fully responsive interface that works on desktop and mobile devices
- **Real-time Updates**: Automatically sync your latest achievements from connected platforms

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/coder-detail.git

# Navigate to the project directory
cd coder-detail

# Install dependencies
npm install

# Start the development server
npm run dev
```
